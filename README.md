## Description

Better Bank Web is a simple web application that provides a basic banking interface. It allows users to view account information, perform transactions, and manage their banking needs through an intuitive web interface.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS
- **Interactivity**: Vanilla JavaScript

## Installation

### Option 1: Visit Live Website

You can also access the application directly at [better-bank-web.vercel.app](https://better-bank-web.vercel.app)

### Option 2: Run Locally

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/better-bank-web.git
   ```
2. Navigate to the project directory:
   ```
   cd better-bank-web
   ```
3. Open with Live server for the file `index.html` in the IDE.

No additional dependencies or build steps are required.

## Usage

1. Navigate through the banking interface before clicking on the "Open account" button.
2. Log in by filling the information, and you will be redirected to another website.
3. View account balances, transaction history, and perform basic banking operations.
